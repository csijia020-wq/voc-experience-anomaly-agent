## 一、产品大图

### 1.1 产品定位

**异动分析Agent是VoC回声系统的核心功能模块，通过多Skill编排实现"数据查询→智能计算→报告生成→自动推送"的全流程自动化，帮助业务快速定位体验问题。**

```

│                        单次分析业务流程                                      │

│  ┌─────────────────────────────────────────────────────────────────────┐   │

│  │ 1. 业务名称映射数据集ID                                               │   │
│  │    • 到餐客服 → dacan_cs_wanfu_weekly                                 │   │
│  │                                                                       │   │
│  │ 2. 时间区间计算                                                       │   │
│  │    • 本期：2026-03-16 ~ 2026-03-22                                    │   │
│  │    • 对比期：2025-03-17 ~ 2025-03-23                                  │   │
│  │                                                                       │   │
│  │ 3. 并发查询                                                           │   │
│  │    • 本期周汇总数据                                                   │   │

│  │    • 本期日明细数据                                                   │   │
│  │    • 对比期日明细数据                                                 │   │
│  │                                                                       │   │
│  │ 4. 口径校验                                                           │   │
│  │    • |本期订单量 - 对比期订单量| / 对比期订单量 > 20% → 暂停确认      │   │
│  │                                                                       │   │
│  │ 5. 维度可用性检测                                                     │   │
│  │    • 检查各维度是否有数据                                             │   │

│    │                                                                        │
│    ↓                                                                        │

│  │                                                                       │   │
│  │ 1. 整体万服YoY计算                                                    │   │
│  │    • 万服 = 服务量 ÷ 订单量 × 10000                                   │   │
│  │    • YoY = (本期万服 - 对比期万服) ÷ 对比期万服 × 100%                │   │
│  │                                                                       │   │
│  │ 2. 各维度逐项计算                                                     │   │
│  │    • delta（绝对差）                                                  │   │
│  │    • YoY%（同比增长）                                                 │   │
│  │    • 服务量变化占比（与整体异动的相对关联比例）                                  │   │
│  │    • 次/万单贡献（绝对贡献）                                          │   │
│  │                                                                       │   │

│  │    • disappeared（消失维度）                                          │   │
│  │    • new_actual（近零增长）                                           │   │

│  │    • 7日数据（本期 + 对比期）                                         │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
```
