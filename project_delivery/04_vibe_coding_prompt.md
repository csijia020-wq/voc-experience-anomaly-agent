## Vibe Coding Prompt（复制给 AI）

```
请帮我生成一个 AI 产品的高保真 HTML/CSS/JS 原型，用于作品集展示。

项目信息：
- 项目名称：VoC回声系统 - 体验异动分析Agent
- 产品定位：体验异动分析Agent，通过多Skill编排实现"数据查询→智能计算→报告生成→自动推送"的全流程自动化，帮助体验运营人员快速定位体验问题
- 核心功能：
  1. 单次分析：用户输入"生成到餐客服上周周报"，Agent自动完成全流程
  2. 定时任务：创建定时任务，每周自动生成报告并推送
  3. HTML可视化报告：生成包含6个区域的图表化报告
  4. Markdown文本报告：生成7个模块的文本报告
  5. 邮件推送：自动发送报告链接到用户邮箱
- 目标用户：体验运营人员、业务负责人、产品经理、数据分析师

要求：
1. 用 HTML + Tailwind CSS（可以用 CDN）+ JavaScript 实现
2. 要有清晰的视觉层次和专业的设计风格（参考美团/企业级数据产品风格）
3. 包含以下核心页面/模块：
   - 首页/对话界面：用户输入自然语言命令，Agent执行分析
   - 报告展示界面：HTML可视化报告，包含核心指标卡片、分析结论、多维度Tab
   - 定时任务管理界面：创建/查看/修改/暂停/删除定时任务
   - 执行日志界面：展示Agent执行过程和状态
4. 要有交互动效（按钮悬停、页面切换、加载状态、Toast提示等）
5. 要能展示 AI 的工作过程（如：输入 → 加载中 → 报告生成完成）
6. 响应式设计，支持 PC 和手机
7. 代码结构清晰，注释完善

技术栈：
- HTML5
- Tailwind CSS（可以用 CDN：https://cdn.tailwindcss.com）
- 原生 JavaScript（不需要框架）

设计规范：
- 主色调：#E8372A（红色，用于上升/重要数据）
- 辅色调：#4A90D9（蓝色，用于下降）
- 背景色：白色或浅灰色
- 卡片阴影：shadow-lg rounded-lg
- 字体：系统默认字体，清晰易读

Mock数据说明（用于原型展示）：
- 业务名称：到餐客服、闪购客服、企客业务
- 核心指标：本期万服 12.5、对比期万服 11.8、YoY +5.9%
- 分析结论：本周万服小幅上涨，主要受二线城市和配送问题维度推高

请直接生成完整的可运行代码，放在一个或多个 HTML 文件中。

重要：这是模拟项目，用于作品集展示，不需要真实的后端 API。
```
