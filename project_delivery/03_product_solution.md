## 一、产品大图

### 一句话产品

**体验异动分析Agent是VoC回声系统的核心功能模块，通过多Skill编排实现"数据查询→智能计算→报告生成→自动推送"的全流程自动化，帮助体验运营人员快速定位体验问题，节省2-3小时/周。**

### 核心价值

- **效率提升**：从人工分析2.5-3.5小时 → Agent分析3分钟，效率提升50-70倍
- **准确性保障**：自动计算YoY、服务量变化占比等指标，避免人工计算错误
- **标准化输出**：统一报告格式，便于横向对比不同业务线
- **自动化运营**：定时任务自动推送，减轻运营负担

## 二、核心功能模块

### 模块 1：单次分析

- **功能描述**：用户通过自然语言输入业务名称和时间范围，Agent自动执行分析并返回报告
- **用户价值**：
  - 无需记忆数据集ID，业务名称自动映射
  - 一键生成完整周报，3分钟完成
  - 支持同比（YoY）和环比对比
- **AI 如何应用**：
  - 意图识别：理解用户输入的业务名称和时间范围
  - Skill编排：动态调用数据查询→计算→报告生成链路
  - LLM生成：自动生成分析结论和处置建议

### 模块 2：定时任务

- **功能描述**：用户创建定时任务，Agent每周自动生成报告并推送
- **用户价值**：
  - 一次创建，每周自动执行
  - 无需每周手动操作
  - 定时推送，确保及时送达
- **AI 如何应用**：
  - Cron调度：定时触发分析任务
  - 任务管理：创建/查看/修改/暂停/删除任务
  - 自动推送：报告生成后自动发送到用户邮箱

### 模块 3：HTML可视化报告

- **功能描述**：生成HTML可视化报告，上传至CDN，返回永久链接
- **用户价值**：
  - 可视化展示，数据一目了然
  - 支持多维度Tab切换
  - 永久链接，便于分享
- **AI 如何应用**：
  - 模板渲染：基于HTML模板生成报告
  - 数据可视化：图表、卡片、趋势图
  - CDN上传：自动上传到S3Plus，返回永久链接

### 模块 4：Markdown文本报告

- **功能描述**：生成Markdown文本报告，包含7个模块完整内容
- **用户价值**：
  - 文本格式，便于复制和编辑
  - 结构化内容，逻辑清晰
  - 7个模块覆盖全面分析需求
- **AI 如何应用**：
  - 结构化生成：7个模块标准结构
  - LLM生成：自动撰写分析结论
  - 格式统一：标准化输出格式

### 模块 5：邮件推送

- **功能描述**：自动发送报告链接到用户邮箱
- **用户价值**：
  - 无需手动发送
  - 邮件直达，确保收到
  - 包含执行摘要，快速了解
- **AI 如何应用**：
  - 邮件生成：包含标题、正文、链接
  - 自动发送：任务完成后自动触发
  - 失败重试：发送失败自动重试

## 六、交互设计要点

### 关键交互

- **单次分析交互**：用户输入"生成到餐客服上周周报" → Agent自动完成全流程 → 返回报告链接
- **定时任务交互**：用户输入"每周一9点生成到餐周报" → Agent确认参数 → 创建成功
- **任务管理交互**：用户输入"查看/修改/暂停/删除任务" → Agent执行 → 返回结果

### 反馈设计

- **执行中反馈**：
  - "正在查询数据，请稍候..."
  - "正在计算指标，请稍候..."
  - "正在生成报告，请稍候..."
- **成功反馈**：
  - "报告已生成！链接：xxx"
  - "定时任务创建成功！每周一9点执行"
- **错误反馈**：
  - "数据查询失败，正在重试..."（重试中）
  - "数据查询失败，请检查网络后重试"（重试失败）
  - "口径差异过大（20%），请确认是否继续"
