## 八、演示稳定性说明

### 8.1 Demo 稳定模式

当前演示默认启用稳定模拟数据：

```env
DEMO_SEED=20260702
DEMO_DETERMINISTIC=true
```

相同业务和时间范围会生成相同模拟数据，方便录屏和面试讲解；更换业务、时间范围或 Demo Seed 会生成不同数据。模拟数据仅用于作品集演示，不代表真实业务。

### 8.2 服务量变化占比

对外统一使用“服务量变化占比”：

```text
服务量变化占比 = 某因素服务量变化 / 本期整体服务量
```

该指标用于近似衡量因素变化与整体异动之间的相对关联，不代表严格因果关系，也不是精确归因值。旧字段 `contrib_wanfu` 暂时保留兼容，新字段为 `service_change_ratio`。

### 8.3 DeepSeek 降级机制

实时模型不可用时，系统仍会完成模拟数据查询和指标计算，并生成结构化降级报告。降级报告中的数字均来自程序计算结果。可通过以下命令诊断配置、DNS、TCP、TLS、鉴权和模型调用：

```powershell
python scripts/check_deepseek_connection.py
```

### 8.4 启动检查

启动后建议先执行：

```powershell
python start_servers.py check
```

脚本会确认 8000 端口上的 OpenAPI 标题为 `VoC 体验异动分析 Agent`，并检查 `/api/chat/stream` 是否存在。若 8000 被其他项目占用，启动脚本会明确报错，不会自动结束未知进程。
